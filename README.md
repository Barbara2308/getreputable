## 🔍 What is getreputable?

getreputable is a tool that calculates reputation scores for autonomous AI agents. It uses a system built on Bittensor, a technology that helps AI models connect and work together. This software aims to give a clear and reliable way to measure how trustworthy an AI agent is.  

Reputation scores can help decide which AI models to trust when they interact or share their work. getreputable focuses on making this process simple, accurate, and dependable.
